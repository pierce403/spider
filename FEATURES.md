# Features

- SDR-first Android workflow for TPMS capture
- USB support for RTL-SDR and HackRF One
- TCP bridge support for remote `rtl_433` streams
- TPMS protocol parsing with vendor lookup hints
- Local sensor history with sighting rollups
- Search, sort, live-only, starred-only, and battery-low filters
- Raw JSON export from the detail screen
- Diagnostics screen with state, counters, and recent logs
